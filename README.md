# GoalGuru AI 🚀

GoalGuru AI is an intelligent productivity and career guidance platform that helps users plan goals, track progress, manage tasks, and receive AI-powered recommendations for personal and professional growth.

## Features
- Goal Tracking
- AI Productivity Suggestions
- Smart Reminders
- Progress Analytics
- Career Guidance
- Daily Task Management

## Tech Stack
- React
- Node.js
- MongoDB
- Gemini API
