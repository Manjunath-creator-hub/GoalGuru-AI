# GoalGuru AI 🚀

GoalGuru AI is an AI-powered productivity and career guidance platform designed to help users manage goals, improve productivity, and accelerate professional growth. The platform provides intelligent features such as AI-generated cover letters, resume building, mock interview preparation, career analytics, and personalized recommendations.

## Features

* AI Resume Builder
* AI Cover Letter Generator
* Mock Interview Preparation
* Career Analytics Dashboard
* Productivity Tracking
* Personalized Career Recommendations

## Technologies Used

* Next.js
* React.js
* Node.js
* Prisma
* PostgreSQL
* Gemini API / OpenAI API
* Tailwind CSS
* Git & GitHub

## Installation

```bash
npm install
npm run dev
```
